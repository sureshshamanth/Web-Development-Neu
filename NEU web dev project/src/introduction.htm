Introduction:

My project will be a website that gives details about a Museum,
it serves as a platform where the users can view the different artifacts of the museum and also get directions and help to visit the museum.
The website acts as a place for museum members to express their views and ideas about their visits,
which the other members can read about and get an idea of how the exhibits are and decide on visiting the museum.
The website also provides a platform for users to buy tickets based on the type of tickets they are elligible for.

Objectives:

The objectives of the project is to provide the user with an interactive website where data is retrieved from an API and displayed as per the users request.
Provide the user access to the best technology by simplyfying it and provide it in a way the user enjoys the experience.
Provide a user interface that looks and feels really good using the best styling possible.
