Layout Design
Layouts are one of the most key aspects for a website and hence I have used the best layout designs for my projects.
I have used BOOTSTRAP throughout my project for each and every page. Benefits of using Bootstrap:
Designs the website to perfection and also very easy to make the website tablet and moblie friendly by adding very minimal javascripts and jQuery.
Requires writing very little code where the bootstrap CSS and javasrcript does all the layout work to perfection.
API Handling
Thanks to the Brooklyn museum I was able to find an awesome API best suited for this project: BROOKLYN MUSEUM API
I had to make use of the YQL which helps in processing the type of different API responses into the format desired by us which can be used for further processing The API response is a JSON response which is in the form of JSON objects and arrays which need to be handled in the right way to make use of them
For the maps page in the website I have made use of the Google Maps API which is a wonderful API to explore and implement
Using the Google maps API helps users who have trouble locating the museum to find the museum with ease and also help users to plan their visit
Login module
I have implemented the Login module using the ASP .net login feature
After successfully connecting to the Database and creating the user login table I have created users who can login with their credentials and leave their valuable comments
Upon leaving the comments the comments of the users will be stored in an XML flat file usnig the XML write features: MSDN
Once the user is done providing the comments he can log off
