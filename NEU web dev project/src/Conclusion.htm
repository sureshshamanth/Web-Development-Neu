Project Conclusion and Future Enhancements

Museum Locator

The Visit page of the project has met the requirements that the user will need from a maps page

Some of the future enhancements that can be suggested for this page are:
Provide multiple map views so that the page can accomodate more than one map.
Can also suggest nearby museums and attractions in New York city that the user would enjoy.

User Feedback Page

The Feedback page of the project meets the requirement for the user to provide feedback

Some of the future enhancements that can be suggested for this page are:
Provide options for users to chat online with museum representatives so that they can have queries resolved without having to call the museum
Give the user an option to rate his reviews on a scale from 1 to 5

Search Module

The Search page of the project meets the requirement for the user to search for information regarding artifacts

Some of the future enhancements that can be suggested for this page are:
Make searching easier by categorizing the artifacts and instead of user typing in a query they can just search based on category
Help users while typing queries by providing autocomplete feature

Tickets Shopping Cart

The tickets page of the project meets the requirement for the user to purchase tickets for the museum

Some of the future enhancements that can be suggested for this page are:
Have a currency converter since majority of the visitors will be foreigners
Save cart details of ticket cart so that the next time the user log in they can continue with purchase
